# design-docs/superseded/

Reversed decisions, kept as record — the rationale that *was* current truth, retired when the decision changed.

- **Status: superseded / historical.** Retained deliberately: a reversed decision is evidence — it explains why the current choice exists and what was already tried. Don't delete; don't garden.
- **Provenance.** Each doc carries a stamp — `STATUS: superseded · <what was chosen, when, and what replaced it> · kept as record`.
- **Naming:** topic-named, like all design-docs (the date lives in the stamp, not the filename).
