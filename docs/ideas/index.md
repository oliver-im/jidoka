# ideas/

Open-ended explorations and problem analysis — the "what if / why / should we" bucket, **before** anything is committed to as scoped work.

- **Status: open.** Contents here **may drift** from the codebase; the folder name is the disclaimer. Nothing in `ideas/` is a promise, and an agent should read a file here as thinking-in-progress, not current truth.
- **Lifecycle.** An idea graduates to *either* an execution plan (`../exec-plans/active/`) *or* a settled decision (`../design-docs/`) — or it gets pruned. It is a peer of both, not a sub-state of either.
- **Naming:** `YYMMDD-N-slug.md` — a dated file sharing the daily counter `N` with `exec-plans/`. When an idea graduates to a plan it **keeps its id**, becoming a directory (`260607-3-foo.md` → `../exec-plans/active/260607-3-foo/`).
